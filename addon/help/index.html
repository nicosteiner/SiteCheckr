Settings

Load ruleset

Here you can import a ruleset-file. Please check the import by clicking “open current”.

Please note: Currently there is a bug when trying to reload a ruleset file. Please delete your existing ruleset ("use default ruleset") before reimporting it.

Open current ruleset

To have a look at the ruleset used for checking your websites click the “open current”-Button.

Use default ruleset

Deletes (!) all imported ruleset data and imports the default ruleset file shipped with SiteCheckr.

Check pages on visit

Enables automatic checking of websites, when you visit them. Please also adjust the following setting!

Only check this pages on visit

Previous setting has to be enabled!
List all URLs, which should be automatically checked, separated by “|” (e.g. http://de.sitecheckr.org/*|http://en.sitecheckr.org/*)
You can find information on the allowed pattern syntax in the addon sdk docs.
Please note: Avoid checking all pages on visit (with a simple “*”) because of performance issues.
